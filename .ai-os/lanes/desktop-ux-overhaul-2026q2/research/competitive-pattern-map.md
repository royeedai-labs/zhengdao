# Competitive Pattern Map

## Portable Patterns

| Source | Pattern | Zhengdao usage |
| --- | --- | --- |
| Linear Command Menu | View/selection-aware command menu | 命令面板按书架、章节、选区、角色、设定等上下文显示动作 |
| Linear Peek | Preview without navigation | 章节、角色、设定、AI 草稿、检查问题支持预览，不强迫离开编辑器 |
| Linear Display Options | Per-view display preferences | 书架网格/列表、工作区布局、角色/设定列表视图持久化 |
| Raycast Root Search | Single keyboard-first action entry | 命令面板成为动作入口；空查询显示高频动作和最近上下文 |
| Raycast Action Panel | Actions for selected item | 章节、角色、草稿、搜索结果、警告项有统一上下文动作面板 |
| Raycast Favorites/Aliases | Fast access to repeated actions | 高频写作动作支持快捷键/别名，低频动作不占顶栏 |
| Notion Slash Commands | Low-friction in-editor creation | 编辑器内逐步引入 slash/mention 风格动作，但不破坏正文输入 |
| Arc Spaces/Sidebar | Context zones | 工作区布局以写作、审阅、设定、专注等任务空间组织 |

## Non-Goals

- 不照搬任何产品的视觉皮肤。
- 不把 Zhengdao 改成纯命令启动器、纯聊天产品或浏览器式多标签产品。
- 不牺牲本地写作主路径来展示 AI 功能。

## UX Quality Bar

- 页面第一眼能判断“我现在在哪、能做什么、下一步最可能做什么”。
- 主操作唯一且稳定，危险操作永远需要确认。
- 键盘路径和鼠标路径都完整；快捷键不需要记住也能被发现。
- 空状态给下一步，不讲产品介绍。
- 错误状态说清原因和可恢复动作。
