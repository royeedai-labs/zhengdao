# 变更请求

- **变更标题**：Windows 安装版系统托盘图标补齐
- **提出时间**：2026-04-23 23:21:58 +08:00
- **变更原因**：用户在 Windows 安装完成并运行后，未在右下角通知区域看到应用图标。静态排查确认当前仅配置了窗口、安装器和快捷方式图标，主进程没有创建 Electron `Tray`，关闭全部窗口后也会直接退出。
- **优先级**：P1

## 变更内容

- [新增] Windows 运行态创建系统托盘图标，复用正式品牌图标资源
- [新增] 托盘菜单提供“显示证道 / 隐藏到托盘 / 退出证道”
- [新增] Windows 点击主窗口关闭按钮时隐藏到托盘，显式“退出证道”或应用退出流程仍真正退出
- [新增] 托盘规则补项目原生测试 guard

| memory | 否 | 当前不回写共享 memory |

## 新增风险 / blocker

- 关闭按钮语义从“直接退出”变成“隐藏到托盘”，必须保留托盘菜单显式退出
- 托盘图标只能在应用运行后出现；不做 Windows 任务栏自动固定
- Windows 真实安装环境仍需人工确认通知区域是否被系统折叠到隐藏图标面板
