# 变更请求：角色关系图谱名称持续下移修复

- **变更标题**：修复角色总库关系图谱内人物名称持续下移
- **提出时间**：2026-04-22 21:57:09 Asia/Shanghai
- **变更原因**：用户反馈角色总库的人物关系图谱中，人物名称一直往下掉，观感像滚动条持续自动滚动。
- **优先级**：P0

## 问题与根因

- **复现路径**：打开作品 -> 顶部「角色总库」->「关系图谱」。
- **根因**：`RelationGraph` 容器只有最小高度，内部 canvas 高度由 React state 写回；ResizeObserver 又读取包含边框/布局反馈后的容器高度并再次写入 state，形成尺寸反馈循环。
- **用户影响**：图谱节点和人物名称持续向下偏移，关系图谱不可稳定阅读。

## 变更内容

- 固定关系图谱容器的布局高度，让 canvas 绝对填充容器，不再反向撑开容器。
- ResizeObserver 使用内容区域尺寸并跳过未变化尺寸更新，避免重复渲染反馈。
