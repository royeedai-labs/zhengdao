# 变更记录：AI 对话会话管理与流式输出

- **基线 ID**：CR-20260422-003456-chat-session-streaming
- **时间**：2026-04-22 00:34
- **类型**：change
- **影响级别**：P1

## 变更原因

用户补充 AI 对话框体验要求：

1. AI 对话框需要支持清空聊天记录。
2. AI 对话框需要支持创建新会话。
3. AI 输出需要流式输出并实时渲染，而不是等完整结果返回后一次性显示。

## 变更内容

1. 新增 AI 会话管理能力：
   - 创建新会话
   - 列出作品会话
   - 切换会话
   - 清空当前会话消息与当前会话关联的待处理草稿
2. AI 助手发送消息后立即显示 assistant 临时消息，并随 token 到达实时追加内容。
3. 流式完成后再持久化 assistant message，并解析草稿篮。
4. Gemini CLI 需要补充 `stream-json` 输出解析与主进程流式桥接；OpenAI / Gemini API / Ollama 复用现有 renderer stream adapter。

## 风险与约束

- 清空聊天记录属于本地删除动作，必须在 UI 里二次确认。
- 清空当前会话只删除 AI 对话消息和该会话关联的草稿，不回滚已应用到小说正文或资产的数据。
- 流式输出期间取消/失败时，不能写入半成品草稿。
